## 本日のお題（Python）

### **お題: 単語出現カウントツール**

テキストファイルを読み込み、
各単語の出現回数を数えて、出現回数の多い順に表示してください。

### 条件

1. 入力は `input.txt` というテキストファイル。
2. 記号（,.!?など）は無視して、単語だけをカウント。
3. 出力は「単語: 出現回数」という形式。
4. 出現回数の多い上位10個を表示。

### 実行例

`input.txt` が以下の内容の場合:

```
This is a pen. This pen is blue. That pen is red.
```

出力:

```
is: 3
pen: 3
this: 2
a: 1
blue: 1
that: 1
red: 1
```
